	\section{Citazioni e riferimenti}
	\label{sec:citations-it}
	Presentiamo due approcci distinti per citare le voci della bibliografia. Il primo metodo prevede citazioni nel testo usando \texttt{\textbackslash citet\{CHIAVE\}} (es. Rossi 2024), mentre il secondo metodo impiega \texttt{\textbackslash citep\{CHIAVE\}} per citazioni tra parentesi (es. [Rossi 2024]). Puoi citare più lavori nello stesso ambiente usando \texttt{\textbackslash citep\{CHIAVE1, CHIAVE2\}}.
	
	Come per le citazioni bibliografiche, è consigliabile usare i riferimenti incrociati nel documento per richiamare capitoli, sezioni, figure o tabelle. Per fare ciò, crea prima un'etichetta con \texttt{\textbackslash label\{NOME\}} all'interno dell'elemento che vuoi richiamare. Successivamente, usa \texttt{\textbackslash autoref\{NOME\}} nel testo. Questo comando crea automaticamente un link cliccabile con la parola corretta, ad esempio: \autoref{cp:latex-tutorial}.
	
	\begin{block}[tip]
		Citazioni e riferimenti incrociati corretti sono vitali nella scrittura accademica. Assicurano credibilità, trasparenza e facilitano la navigazione del documento da parte della commissione di laurea.
	\end{block}

	\section{Figure}
	In \LaTeX, integrare le figure è un processo standardizzato. Si utilizza l'ambiente \texttt{\textbackslash begin\{figure\}}. Puoi personalizzare il parametro \texttt{width} in base alle tue esigenze ed è fondamentale fornire una didascalia ben scritta. L'ambiente della didascalia è denotato da \texttt{\textbackslash caption\{TESTO\}}. Per generare una didascalia più corta per l'indice delle figure, utilizza il formato \texttt{\textbackslash caption[TESTO\_BREVE]\{TESTO\_LUNGO\}}.

	\section{Equazioni e formule matematiche}
	Quando si scrivono equazioni, \LaTeX\ non ha rivali. Puoi inserire una formula all'interno del testo usando i delimitatori \texttt{\textbackslash( \ldots\ \textbackslash)}, oppure usare \texttt{\textbackslash begin\{equation\}} per centrarla nella pagina e numerarla in automatico. Se preferisci non numerare l'equazione, usa i delimitatori \texttt{\textbackslash[ \ldots\ \textbackslash]}.
	
	\vspace{.875em}
	\textbf{Esempio:} In fisica, l'equivalenza massa-energia è espressa dall'equazione \(E=mc^2\), scoperta da Albert Einstein. Nell'equazione (\ref{eq:equation-01-it}) ne vediamo la forma base:
	
	\begin{equation}
		\label{eq:equation-01-it}
		E=mc^2
	\end{equation}
	
	Anche per le equazioni, se non conosci a memoria i comandi matematici, ti consigliamo di usare un editor online visuale come \href{https://latexeditor.lagrida.com/}{Lagrida LaTeX Equation Editor}.

	\section{Citations and references}
	\label{sec:citations-en}
	We present two distinct approaches for citing bibliography entries. The first method involves in-text citations using \texttt{\textbackslash citet\{ENTRY\}}, while the second method employs \texttt{\textbackslash citep\{ENTRY\}} for citations within brackets. You can cite multiple works at once using \texttt{\textbackslash citep\{ENTRY1, ENTRY2\}}.
	
	Much like citations, it is advisable to use cross-references in your document when referring to chapters, sections, figures, or tables. To do so, first create a label using \texttt{\textbackslash label\{TEXT\}} inside the element you want to reference. Then use \texttt{\textbackslash autoref\{TEXT\}} in the text. This command automatically creates a clickable link with the correct word, for instance: \autoref{cp:latex-tutorial}.
	
	\begin{block}[tip]
		Proper citations and cross-references are vital in academic writing. They ensure credibility, transparency, and make the document easier to navigate for the graduation committee.
	\end{block}

	\section{Tables}
	Tables are essential for presenting findings clearly. This template simplifies the process by providing pre-configured styles.
	
	\begin{block}[tip]
		All table environments should use the \texttt{[!htpb]} float options to allow \LaTeX\ to find the best placement on the page, avoiding excessive white space.
	\end{block}
	
	The conventional \texttt{\textbackslash begin\{tabular\}} environment enables you to create simple yet elegant tables. \autoref{tab:table-01-en} uses the \texttt{booktabs} commands (\texttt{\textbackslash toprule}, \texttt{\textbackslash midrule}, \texttt{\textbackslash bottomrule}) for a clean style.

	\section{Equations and mathematics}
	When writing equations, \LaTeX\ is unrivalled. You can insert a formula inline using the delimiters \texttt{\textbackslash( \ldots\ \textbackslash)}, or use \texttt{\textbackslash begin\{equation\}} to display it centred with automatic numbering. If you prefer no numbering, use the delimiters \texttt{\textbackslash[ \ldots\ \textbackslash]}.
	
	\vspace{.875em}
	\textbf{Example:} In physics, mass-energy equivalence is expressed by \(E=mc^2\). In equation (\ref{eq:equation-01-en}) we see its basic form:
	
	\begin{equation}
		\label{eq:equation-01-en}
		E=mc^2
	\end{equation}
	
	If you are not familiar with mathematical commands, we recommend using an online visual editor like \href{https://latexeditor.lagrida.com/}{Lagrida LaTeX Equation Editor}.
